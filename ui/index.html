Welcome To Easy Diffusion

Stable Diffusion uses "models" to generate images. There are various different models available. To start, choose the base models to download. If unsure, use the default settings. You can add more models later on.

1.4
The original base model.
1.5
An updated version of SD 1.4 with slight improvements.
2.1
SD 2.1 was trained on a dataset of images with a resolution of 512√ó512 with NSFW images removed.
2.1 768
SD 2.1-768 was trained on a dataset of images with a resolution of 768√ó768 with NSFW images removed. (6GB of GPU Video RAM recommended)

Instead of the base models you can also use other models in ckpt or safetensors format. Place them into the models/stable-diffusion folder in the installation directory.
